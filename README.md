# body-recognition-py

該程序可以擷取電腦螢幕中的影像進行肢體識別

使用 mediapipe 寫出來的肢體辨識

請先安裝 python

請按照以下指令下載

```
git clone https://github.com/MHNightCat/body-recognition-py
cd ./body-recognition-py
```

請按照以下指令安裝

```
pip install cv2
pip install mediapipe
```

安裝完後使用

```
py index.py
```
